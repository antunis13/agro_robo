## Como rodar o projeto

Baixem o python 3.12

Depois rodem os comandos:

```bash
    python3 -m venv venv
    venv\Scripts\activate
    python3 -m pip install pygame
```

ou se estiver no ambiente macOS

```bash
    python3 -m venv venv
    source venv/bin/activate
    python3 -m pip install pygame
```
